# CustomerComplaintAnalysis

This project was done in CoLab with assistance from Gemini. The goal was to analyze which customers were filing the most complaints, where the complaints were targeted, and through which channel contianed the highest volume of complaints. 

# Gemini's Influence

I utilized Gemini for writing the code while coaching it through useful methods. I noticed that Gemini often only performed surface level EDA and Pre-Preocessing without using the context to build deeper insights.
